The easiest way to fix the problem is to:

Use Google Chrome or Mozilla Firefox


If you want to use Internet Explorer instead, all you need to do is download Google Chrome Frame from Google.


If you want to use Safari instead (& have a Mac), you need to fix it as follows:
-Go to Safari in the top left corner, then preferences.
-Go to the advanced tab, and then check "Show Develop Menu in menu bar" at the bottom.
-Now close it, go to Develop, and then click "Enable WebGL" at the bottom.
-Refresh the page, and it should now work.
Note that this only works for macs: PC users must use another browser.
